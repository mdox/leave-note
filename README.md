# Leave Notes

Web App for centralized place for text notes for multiple users.

# Model

Web <-> Server <-> Middleware <-> Controller <-> Service <-> DAO <-> Database

# Dependency Programs

- npm
- nodejs
- docker
- docker-compose
- postgresql (dockerized!)

# Dependency Utilities

- vite
- react
- react-router-dom
- tailwindcss
- pg (postgresql api)
- knex (database api)
- express
